# Box Drawing Art
Use Box-drawing character to make sentence.

# Usage
## Java
```java
System.out.println(BoxDrawingArt.convert("Box Drawing Art"));
```

## Kotlin
```kotlin
println("Box Drawing Art".toBoxDrawingArt())
```
